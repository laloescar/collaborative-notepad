Collaborative Text Editor Registration Interface Description
This document provides a structured, technical blueprint of the registration screen image.

1. High-Level Layout & Structure
- Layout Type: Split-pane container (2-column layout on desktop).
- Aspect Ratio: 4:3 horizontal aspect ratio container.
- Proportions:
	- Left Column (Promotional Card): ~40% width.
	- Right Column (Form Panel): ~60% width.
- Spacing: Generous padding (p-8 to p-12), clean whitespace, aligned grids.

2. Global Design Tokens (Theme)
- Colors: follow design guidelines.
- Font Family: follow design guidelines.

3. Left Panel: Promotional Card
- Background: Light Blue.
- Padding: p-10.
- Top Asset (Logotype): Currently non existing.
- Typography (follow design guidelines):
	- Title: Bold, dark blue, centered: Únete junto a tus amigos a CoopDoc.
	- Short Divider: Tiny horizontal accent bar below the title (light blue, width: w-12, height: h-1).
	- Description: Center-aligned, slate gray text: Usando CoopDoc, comparte notas de volada. Para ello, crea una cuenta nueva con nosotros.
- Illustration (might not exist in the end):
	- Located in the lower-middle section.
	- Vector art featuring three team members.

4. Right Panel: Form Panel
- Padding: p-12.
- Header:
	- Main Title: ¡Bienvenido! (Bold, large size, dark slate text, follow design guidelines).
- Form Grid: Stacked inputs with a vertical gap of space-y-4. Each field follows this blueprint:
	- Label: Small, bold, dark slate text above the input.
	- Input Container: Relative container, height: h-12.
	- Placeholder text: Italicized gray text starting with "ej." to guide the user.
- Right Button (Password fields only): An interactive button to toggle password visibility.
- Field-by-Field Breakdown
	- Name & Surname:
		- Label: Ingresa tu nombre completo:
		- Placeholder: ej. Nombre Apellido Apellido
	- Date of Birth:
		- Label: Ingresa tu fecha de nacimiento (dd/mm/aaaa):
		- Placeholder: ej. 17/09/2001
	- Verification Email:
		- Label: Ingresa tu correo electrónico:
		- Placeholder: ej. usuario@correo.com
	- Password:
		- Label: Crea tu contraseña:
		- Right Button: Password visibility toggle (label: Show).
		- Placeholder: ej. Contraseña1!
	- Confirm Password:
		- Label: Verifica tu contraseña:
		- Right Button: Password visibility toggle (label: Show).
		- Placeholder: Repite tu contraseña

5. Primary Actions & Authentication
- Primary Button:
	- Text: Crear cuenta
	- Styles: Solid background, text color (follow design guidelines), height h-12, width w-full. Hover state should darken.
- Divider:
	- Horizontal line (h-px bg-slate-200) with a central text badge reading o (lowercase "or" in Spanish) styled with text-slate-400 and padded inline with background color masking.
- Footer Action:
	- Text: Ya tengo cuenta (slate) followed by Regresar a inicio de sesión (blue hyperlink text).
